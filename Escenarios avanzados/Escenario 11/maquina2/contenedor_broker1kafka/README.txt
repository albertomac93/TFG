# Escenario 11. Máquina 2 - Contenedor broker1kafka

Rutas donde almacenar los ficheros suponiendo que el contenedor está en /var/lib/machines/broker1kafka

kafka-broker.service -> /var/lib/machines/broker1kafka/etc/systemd/system/kafka-broker.service
server.properties -> /var/lib/machines/broker1kafka/usr/local/src/kafka/config/server.properties
