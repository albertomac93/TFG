# Escenario 11. Máquina 1 - Contenedor broker0kafka

Rutas donde almacenar los ficheros suponiendo que el contenedor está en /var/lib/machines/broker0kafka

kafka-broker.service -> /var/lib/machines/broker0kafka/etc/systemd/system/kafka-broker.service
server.properties -> /var/lib/machines/broker0kafka/usr/local/src/kafka/config/server.properties
